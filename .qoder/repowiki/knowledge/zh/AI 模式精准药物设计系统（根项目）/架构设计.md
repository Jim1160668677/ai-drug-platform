根目录作为多子系统的编排层：
- 运行时由两个独立进程组成——`backend/app/main.py` 启动 FastAPI/Uvicorn（端口 8000），`frontend/app.py` 启动 Streamlit（端口 8501），两者通过 HTTP REST 调用耦合，前端 `api_client.py` 封装所有 `/api/v1/*` 端点。
- 共享契约集中在 `pyproject.toml`：ruff lint/format 规则、mypy 配置、pytest 收集路径与覆盖率阈值（≥75%）、以及 per-file ignore 列表同时约束 backend 与 frontend；`environment.yml` 用 conda-forge/bioconda/pip 三层 channel 统一锁定 Python 3.11、RDKit、Scanpy、FastAPI、Streamlit 等跨子模块依赖。
- 数据库/缓存/对象存储等外部资源通过 `.env` 环境变量注入，后端 `core/config.py` 读取后供所有 services/models/schemas 复用，形成“配置即边界”的横向解耦。
- 四个业务子系统（A 数据整合 / B 靶点发现 / C 治疗方案 / D 协作平台）按 `services/{analyzer,knowledge,optimizer,guardrail,...}` 目录切分，前端 `pages/01_~10_*.py` 一一对应，通过统一的 JWT RBAC 中间件在 API 层做权限拦截。
