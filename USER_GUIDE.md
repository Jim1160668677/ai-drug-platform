# 用户操作手册

## 一、系统概述

AI模式精准药物设计系统是一个集成多组学数据分析、AI 靶点发现、并行治疗方案设计的精准医疗平台。系统灵感来源于 GitLab 联合创始人 Sid Sijbrandij 的个性化癌症治疗经历。

### 1.1 核心能力

| 子系统 | 功能 |
|--------|------|
| A. 极限诊断数据整合 | RNA-seq/单细胞/VCF/FASTA 数据接入与解析 |
| B. AI 辅助靶点发现 | 靶点识别、分子设计、老药新用、证据链构建 |
| C. 并行治疗方案设计 | 多疗法组合优化、疗效监测、动态调整 |
| D. AI 模式协作平台 | 权限管理、多角色协作、数据看板、合规审计 |

### 1.2 角色权限

| 角色 | 权限范围 |
|------|----------|
| Founder（创始人/患者） | 全部数据 + 模型干预 + 紧急决策 + 管理 |
| Chief Researcher（首席研究员） | 全部分析结果 + 部分原始数据 + 模型配置 |
| Researcher（研究员） | 分配项目内的数据 + 标准分析 |
| Doctor（医生） | 临床数据 + 靶点报告只读 |
| Data Engineer（数据工程师） | 系统日志 + 数据质量指标 |

## 三、功能操作流程

### 3.1 数据上传与解析

1. 进入"数据管理"页面
2. 点击"上传文件"，选择 RNA-seq/单细胞/VCF/FASTA 文件
3. 上传完成后点击"解析"触发自动解析
4. 查看解析结果与质量报告

### 3.2 AI 靶点发现

1. 进入"靶点发现"页面
2. 选择目标项目
3. 点击"发现靶点"，系统自动调用：
   - MyGene/MyVariant 基因变异注释
   - 差异表达分析（单细胞数据）
   - 证据链构建
4. 查看候选靶点列表（含置信度、证据等级）
5. 对特定靶点点击"老药新用"获取已获批药物候选

### 3.3 分子设计与评估

1. 进入"分子库"页面
2. 输入靶点或 SMILES 字符串
3. 点击"设计分子"生成候选分子
4. 查看类药性评估（Lipinski 五规则、溶解度、毒性）

### 3.4 AI 智能问答

1. 进入"AI 问答"页面
2. 选择分析层级：
   - **快速筛查**（fast_screen）：低成本、快速响应
   - **深度洞察**（deep_insight）：RAG + 知识图谱 + 深度分析
3. 输入问题（如"EGFR T790M 突变有哪些治疗策略？"）
4. 查看结构化回答（含引用文献、分析代码、图表）

### 3.5 多假设并行管理

1. 进入"多假设并行"页面
2. 创建新假设（如"靶向 B7H3 + 免疫联合"）
3. 对每个假设独立分析，互不干扰
4. 点击"对比分析"比较多个假设的优劣

### 3.6 干湿闭环

1. 进入"干湿闭环"页面
2. 提交湿实验结果（IC50、存活率等）
3. 系统自动更新预测模型
4. 查看预测值 vs 实测值对比图表

### 3.7 CDISC 报告导出

1. 进入"报告中心"页面
2. 选择项目与报告类型（SDTM/ADaM）
3. 点击"导出"生成符合 FDA 标准的 CSV 文件

## 四、管理后台

### 4.1 用户管理

- 查看用户列表
- 创建/禁用用户账户
- 分配角色

### 4.2 大模型 API 设置

1. 进入"管理后台"页面，找到"大模型 API 设置"卡片
2. 点击"Agnes 预设"一键填充 Agnes 配置
3. 输入 API Key（如 `sk-EGw4...`）
4. 点击"保存"
5. 点击"测试连通性"验证配置
6. 点击"激活"将该配置设为当前使用

支持的参数：
- **名称**：配置标识（如 Agnes、OpenAI、Azure）
- **访问模式**：API Only / 本地部署 / 代理
- **上游协议**：Chat Completions / Completions / Anthropic Messages
- **基础 URL**：API 端点
- **API Key**：密钥（脱敏存储）
- **测试模型**：用于连通性测试的模型名
- **快速/深度模型**：分级路由使用的模型
- **温度/Max Tokens/超时**：生成参数

### 4.3 审计日志

- 查看所有用户操作记录
- 按时间/用户/操作类型筛选

## 五、常见问题

### Q: 如何切换 Mock/Real 模式？
编辑 `backend/.env`，设置 `USE_MOCK=true`（Mock）或 `false`（真实 API），重启后端。

### Q: 忘记密码怎么办？
目前无自助找回功能，需通过管理后台重置或重新注册账户。

### Q: AI 问答响应很慢？
深度洞察模式会调用 RAG + 知识图谱，耗时较长（10-30 秒）。如需快速响应，请选择"快速筛查"模式。

### Q: 数据上传失败？
检查文件格式是否符合要求（RNA-seq: CSV/TSV，单细胞: h5ad，VCF: vcf，FASTA: fa/fasta）。
