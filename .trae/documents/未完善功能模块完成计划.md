  from sqlalchemy import String, JSON, ForeignKey, Text
  from sqlalchemy.orm import Mapped, mapped_column
  from app.models.base import Base, UUIDMixin, TimestampMixin

  class DataLineage(Base, UUIDMixin, TimestampMixin):
      __tablename__ = "data_lineage"
      project_id: Mapped[str] = mapped_column(ForeignKey("projects.id"), nullable=False, index=True)
      source_type: Mapped[str] = mapped_column(String(50), nullable=False)  # dataset/target/molecule/treatment

  from datetime import datetime
  from sqlalchemy import String, DateTime, JSON, ForeignKey, Text
  from sqlalchemy.orm import Mapped, mapped_column
  from app.models.base import Base, UUIDMixin, TimestampMixin

  class ConsentStatus:
      GRANTED = "granted"
      WITHDRAWN = "withdrawn"

      granted_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
      expires_at: Mapped[Optional[datetime]] = mapped_column(DateTime)
      revoked_at: Mapped[Optional[datetime]] = mapped_column(DateTime)
      revoke_reason: Mapped[Optional[str]] = mapped_column(Text)
      purpose: Mapped[str] = mapped_column(Text, nullable=False)
      constraints: Mapped[Optional[dict]] = mapped_column(JSON)  # 约束条件

## 七、Assumptions & Decisions（假设与决策）

### 假设
1. `config.py` 中 `CONSENT_CHECK_ENABLED` 已在功能 1 阶段添加（摘要确认）
2. `guardrail.py` 的 `MedicalRedlineChecker` 已完整实现（摘要确认 71 测试通过）
3. `ddi_checker.py` 已完整实现（摘要确认 26 测试通过）
4. `ws.ts` / `useTaskProgress.ts` / `ProgressBar.tsx` 已创建（探索确认文件存在）
5. 前端 `lib/api/index.ts` 已 re-export `ws`（摘要确认）

### 决策
1. **血缘查询用 BFS 迭代**而非递归 CTE — 兼容 SQLite + PostgreSQL，避免递归深度限制
2. **SDTM 校验器纯 Python 实现** — 不依赖 Java/P21 CLI，8 条核心规则覆盖 FDA 常见问题
3. **同意校验为显式调用**（ConsentManager.check）而非自动中间件 — 避免现有端点的侵入式改造，按需在数据访问层调用
4. **前端 DAG 可视化用 PlotlyChart**（项目已有依赖） — 不引入额外图库（如 react-flow），降低复杂度
5. **功能说明文档放 `docs/features/`** — 集中管理，便于检索
