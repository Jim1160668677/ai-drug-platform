# 七大功能模块增强计划

## Context

用户要求增强 7 个功能模块，覆盖生信分析、多靶点分子设计、干湿闭环、联邦学习、假设生成、一键流水线、统一 AI 大模型架构。前期调研确认所有模块已有基础框架，但各模块实现深度不一：LLM 架构最成熟（缺缓存/版本/配额），分子设计缺多靶点，假设生成缺自动生成，干湿闭环缺临床反馈，联邦学习缺隐私增强，流水线缺假设集成，生信分析缺高级算法。

ERR_ABORTED 已确认为 Next.js 14 开发模式按需编译的瞬时 RSC 请求中止（19 个页面全部返回 HTTP 200，代码无错误），非真实 bug。

## 实施顺序

按依赖关系排序（被依赖的先做）：
1. **功能 7**：统一 AI 大模型架构增强（其他功能的 LLM 调用基础）
2. **功能 1**：生信分析模块增强（产出数据供假设生成）
3. **功能 2**：多靶点协同分子设计（依赖生信分析）
4. **功能 5**：假设自动生成（依赖分析 + 分子设计）
5. **功能 3**：干湿闭环 + 临床反馈（依赖分子设计）
6. **功能 4**：联邦学习增强（依赖实验数据）
7. **功能 6**：一键流水线集成假设（集成所有上述功能）

```python
# hypothesis_generator.py
class HypothesisGenerator:
    """从分析结果自动生成研究假设"""

    async def generate(self, project_id: str, context: dict) -> list:
        """生成假设
        1. 收集前序分析结果（靶点发现、分子设计、差异表达、通路富集）
        2. 规则推理：提取显著上调基因 → 关联已知通路 → 生成假设
        3. LLM 增强：将规则推理结果传入 LLM 生成自然语言假设描述
        4. 结构化输出：假设描述 + 支持证据 + 预期验证方法 + 置信度
        """
        # 规则推理
        rule_based = await self._rule_inference(context)
        # LLM 增强
        llm_enhanced = await self._llm_enhance(rule_based)
        # 结构化
        return self._structure(llm_enhanced)
```

```python
class PrivacyEnhancer:
    def add_dp_noise(self, weights, epsilon=1.0) -> list:
        """差分隐私 — 拉普拉斯噪声注入梯度"""
        sensitivity = max(abs(w) for w in weights)
        scale = sensitivity / epsilon
        return [w + np.random.laplace(0, scale) for w in weights]

    def secure_aggregate(self, client_weights: list) -> list:
        """安全聚合 — 加法掩码保护客户端梯度"""
```

```python
# discovery_pipeline.py 增强
class DiscoveryPipeline:
    async def run(self, config: PipelineConfig) -> dict:
        """执行流水线
        config.steps: list[str] — 用户选择的步骤
        config.inject_hypothesis: bool — 是否自动注入假设
        """
        results = {}
        for step in config.steps:
            if step == "hypothesis_generation":
                # 从前序步骤结果自动生成假设
                results["hypotheses"] = await self._generate_hypotheses(results)
            elif step == "molecule_design":
                # 多靶点设计，靶点来自前序步骤
                targets = results.get("targets", [])
                results["molecules"] = await self._design_molecules(targets)
            # ... 其他步骤
        return results
```
